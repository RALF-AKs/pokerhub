PokerHub — Site Educacional de Poker Texas Hold’em

Projeto desenvolvido para o Projeto Final do Módulo Front-end Estático, com o objetivo de aplicar HTML5 semântico, CSS3, Bootstrap 5 e design responsivo na criação de um site multipágina com tema Poker Texas Hold’em.

Objetivo do Projeto

Criar um website educacional que apresente conceitos fundamentais e avançados do Poker Texas Hold’em, abordando regras, estratégias, gestão de banca e análise de jogadas, utilizando boas práticas de desenvolvimento front-end.

Tecnologias Utilizadas

HTML5 Semântico

CSS3

Bootstrap 5

Google Fonts (Roboto)

GitHub Pages (deploy)

Funcionalidades

Formulário de contato com validação HTML

Layout adaptável para dispositivos móveis

Responsividade

O site se adapta automaticamente para celulares, tablets e desktops utilizando o sistema de grid do Bootstrap, garantindo boa experiência em qualquer tamanho de tela.
